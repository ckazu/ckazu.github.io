ActiveRecord の serialize を使う

rails

ActiveRecord::Base .serialize

rails でオブジェクトをそのまま DB に保存したい場合， serialize が使える場合があります．

class User < ActiveRecord::Base
  serialize :group_ids
end

このようにすると，User#group_ids オブジェクトは serialize されて保存されます...
